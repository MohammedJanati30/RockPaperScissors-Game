🎮 Rock Paper Scissors Game   

A simple desktop game built using C# Windows Forms where the player competes against the computer in the classic Rock Paper Scissors challenge.

🧠 How it works  

The player selects one of the three choices:  

Rock 🪨   

Paper 📄  

Scissors ✂️


The computer randomly selects its choice.  

The winner of each round is determined based on the classic rules:  

Rock beats Scissors
Scissors beats Paper
Paper beats Rock
📊 Features
Multiple rounds gameplay system
Score tracking (Player, Computer, Draws)
Random computer AI selection
Reset game functionality
Visual UI using Windows Forms with images
🛠️ Technologies Used
C#
Windows Forms (.NET Framework)
Visual Studio
🚀 Project Goal

This project was created to practice C# programming, enums, structs, event handling, and building interactive desktop applications.
